# Writeups & Projects

A collection of writeups, notes, and personal lab projects covering CTF challenges and security experiments (TryHackMe, HackTheBox, and others).

## Contents

- **Anonymous_Writeup.pdf** – TryHackMe (Linux, Medium): FTP anonymous login, SMB share enumeration, reverse shell via cron-executed script, SUID `/usr/bin/env` privilege escalation.
- **Attacktive_Directory_Writeup.pdf** – TryHackMe (Windows AD, Medium): Kerberos user enumeration with Kerbrute, AS-REP roasting, SMB share access, DCSync via secretsdump, Pass-the-Hash with Evil-WinRM.
- **arp-spoofing/** – Personal lab project: Man-in-the-Middle attack via ARP spoofing in a virtualized environment (Hyper-V), using arpspoof, Wireshark, and Nmap to intercept traffic between a client and server.

## Disclaimer

These writeups and projects are for educational purposes only. All targets are legal CTF/lab environments or self-hosted virtual machines under the author's control.
